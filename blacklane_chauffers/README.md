View your app in AI Studio: https://ai.studio/apps/drive/10ZPEfCE079-4seM07IZEfuKUASw4lPb4

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
<!-- 2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key -->
<!-- 3. Run the app: -->
   <!-- `npm run dev` -->

2. To Dev
npm run dev

3. To build
npm run build

4. To deploy
- first run build. it will compile all files to dist/
- copy the stuff in dist/ -> to hosting location
